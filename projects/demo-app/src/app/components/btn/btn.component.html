<rp-button> Documentation

The rp-button component is a customizable button with configurable color, size, and theme properties.

Properties

Property Type Possible Values Description
color string | 'default' | 'primary' | 'success' | 'warning' | 'danger' Defines the button color.
size string 'xsmall' | 'small' | 'medium' Defines the button size.
theme string 'square' | 'round' Defines the button shape theme.

Example

 <rp-button 
    [size]="'xsmall'" 
    [colorState]="'default'" 
    [theme]="'round'"
 > Close </rp-button>

Property isDisabled

The isDisabled property disables the button when set to true.

Example

 <rp-button 
    [size]="'xsmall'" 
    [colorState]="'default'" 
    [theme]="'round'"
    [isDisabled]="true"
 > Close </rp-button>
Close Update Insert Delete

confirmDialog

The confirmDialog property accepts an object of type ConfirmDialog. It shows a confirmation dialog to the user before performing the button action.

Example configuration object:


const confirmDelete ConfirmDialog = {
  cancelBtn: 'Close',
  confirmBtn: 'Delete',
  text: 'Are you sure about deleting?',
  title: 'Delete Item',
};

Usage in HTML:

 <rp-button 
    class="inline-block !w-32"
    [size]="'xsmall'" 
    [colorState]="'default'" 
    [theme]="'round'"
    [confirmDialog]="confirmDelete"
 > Delete </rp-button>
