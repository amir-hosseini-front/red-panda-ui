Radio Button

Radio Buttons (Line Type)

This rp-radio-buttons component is used to display a list of options as Radio Buttons.

Properties

  • items: An array of options to display.
  • titleKey: The key in each item that represents the title.
  • valueKey: The key in each item that represents the actual value.
  • defaultValue: The initially selected value. You should provide the value key.
  • layout: Layout of the radio buttons, like row or column.
  • name: The name of the radio button group.
  • onChange: Function called when selection changes; returns the selected value.

Usage Example

<rp-radio-buttons
  class="flex-1"
  [titleKey]="'title'"
  [valueKey]="'value'"
  [layout]="'row'"
  [name]="'r1'"
  [defaultValue]="'RadioButton2'"
  [items]="getFromDestinationFilterList | async"
  (onChange)="onChange($event)"
></rp-radio-buttons>
