rp-selector Component

The rp-selector component is a reusable dropdown selector for Angular applications. It supports dynamic items, default values, and integration with Angular Reactive Forms.

Example Usage

 <rp-selector 
  class="block mx-4 w-80" 
  [titleKey]="'title'" 
  [valueKey]="'value'" 
  [defaultValue]="defaultValue" 
  [label]="'select item'" 
  [items]="gender | async" 
  [inputFormControl]="form.get('city')! | formControl" 
 ></rp-selector>

Properties

Input Type Description
titleKey string The key used for the item’s display label.
valueKey string The key used for the item’s value.
defaultValue T The default selected value.
disabled boolean Disables the input when set to true.
label string Label displayed above the selector.
items any[] The list of items to display in the dropdown.
inputFormControl FormControl Angular Reactive Form control binding.
