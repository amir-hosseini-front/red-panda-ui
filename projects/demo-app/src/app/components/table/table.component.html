Table Component (rp-table)

The rp-table component is designed to display data in a table format. Data is provided via [tableData] and column configuration via [columnsSchema].

Usage Example

<rp-table
  [columnsSchema]="columnsSchema"
  [tableData]="tableData"
></rp-table>

Properties

Property Type Description
columnsSchema Array of ColumnsSchema
tableData Array of any

Column Schema

Each column is defined as an object with the following properties:

  • key: The property name from tableData to display.
  • type: Defines the column type. Supported values include text, number, color, selector, etc.
  • label: The column header text.

Optional Properties

  • copyToClipboard: If true, clicking on a cell will copy its value to the clipboard.
  • editable: If enabled, the user can hold Ctrl and click on a cell to edit its value.

Example Column Schema 

columnsSchema = [
  {
    key: 'fullName',
    type: COLUMNS_TYPES.TEXT,
    label: 'Full Name',
    copyToClipboard: true,
    editable: true
  },
  {
    key: 'age',
    type: COLUMNS_TYPES.NUMBER,
    label: 'Age'
  },
  {
    key: 'favoriteColor',
    type: COLUMNS_TYPES.COLOR,
    label: 'Favorite Color'
  }
];

Editable Columns

The editable property allows users to update table cell values. If editable: true is set in a column’s schema, the user can:

  • Hold Ctrl and Left Click on the cell to enter edit mode.
  • Edit the value directly in the cell.
  • Click the ✔️ Confirm icon to save the changes.
  • When confirmed, the (onColumnValueChanged) event is emitted.

Usage Example

<rp-table
  [columnsSchema]="columnsSchema"
  [tableData]="tableData"
  [loading]="loading"
  (onColumnValueChanged)="columnValueChanged($event)"
></rp-table>

Editable Column Example 

columnsSchema = [
  {
    key: 'fullName',
    type: COLUMNS_TYPES.TEXT,
    label: 'Full Name',
    editable: true
  }
];

Handling Value Changes

The component emits an event when a column value is changed. You can handle it in your component’s TypeScript file: 

columnValueChanged(data: any) {
  console.log('Column value updated:', data);
  // Perform your update logic here (e.g., API call, state update)
}
