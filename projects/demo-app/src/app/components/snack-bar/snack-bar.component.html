Snackbar

Snackbar Component

The Snackbar service provides a simple way to display temporary messages to users. It supports different types such as success, error, info, and warning.

Step 1: Inject the service

Add the SnackbarService into your component's constructor: 

constructor(private snackbar: SnackbarService) { }

Step 2: Show a Snackbar

Use the show method to display a message:

this.snackbar.show({
  message: 'The operation was successful.',
  type: 'success',
  duration: 4000
});

Properties

  • message: The text to display.
  • type: Snackbar style. Possible values: success , error , info , warning
  • duration: The time (in ms) the snackbar remains visible. Default is 3000.

Example

this.snackbar.show({
  message: 'Something went wrong!',
  type: 'error',
  duration: 5000
});
