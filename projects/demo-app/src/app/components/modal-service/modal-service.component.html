Modal Service – onClick() Example

The onClick() method demonstrates how to open a modal dynamically using the ModalService in Angular. It passes custom data to the modal component and listens for modal lifecycle events such as 'Close'.

How It Works

  • When a user triggers an action, the onClick() method is called.
  • The modalService.open() method opens the UserFormComponent and injects initial data (name, email).
  • The service’s modalStatus$ observable emits updates such as 'Close'.
  • When 'Close' is received, modalService.close() is called to close the modal.

Parameters

Parameter Type Description
UserFormComponent Component The Angular component that will be displayed inside the modal.
data object The initial data passed into the modal (e.g. name, email).

Example Usage


onClick() {
  this.modalService.open(UserFormComponent, {
    name: 'Ali Redpanda ',
    email: 'aliredpanda.dev',
  });

  this.modalService.modalStatus$.subscribe((data: string) => {
    if (data === 'Close') {
      this.modalService.close();
    }
  });
}

Notes

  • Ensure ModalService is provided in your Angular module.
  • Always unsubscribe from modalStatus$ in ngOnDestroy() to avoid memory leaks.
  • This modal pattern is ideal for component libraries, providing a clean and consistent API similar to popular UI kits.
