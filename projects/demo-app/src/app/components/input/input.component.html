rp-input Component

The rp-input component is a reusable input field built for Angular applications. It supports common input types, labels, placeholders, and a disabled state.

Properties

Property Type Description
type string Defines the input type.
Accepted values: text, password, number
disabled boolean Disables the input when set to true.
label string Displays a label above the input field.
placeholder string Hint text displayed inside the input field.
formControl FormControl Binds Angular reactive form control to the input.
onChange string Emit value string.

Example Usage

 <rp-input 
  class="flex-1" 
  [type]="'text'"
  label="Email" 
  [placeholder]="'Please enter email'" 
  [formControl]="form.controls.email" 
  (onChange)="onEmailChange($event)"
 > </rp-input>

Input with Custom Validation Messages

The rp-input component supports passing custom validation error messages. You can define validation rules in your Angular form and provide a set of messages through the [errorMessages] property.

1. Define the FormControl with Validators 

  form = this.fb.group({ number: ['', [Validators.pattern(/^\d+$/)]], });

2. Create Validation Messages Array 

  ValidationNumber = [
    { 
      type: 'pattern', 
      pattern: '^\d+$', 
      message: 'Just invalid number', 
    } 
  ];

3. Pass Error Messages to the Component 

  <rp-input 
    [type]="'text'" 
    class="block mx-4 w-60" 
    [placeholder]="'Number'" 
    [errorMessages]="ValidationNumber" 
    [formControl]="form.controls['number']" 
    (onChange)="onChange($event)" 
  ></rp-input>
