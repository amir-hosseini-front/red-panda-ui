Inputs & Outputs

Inputs

Name Type Description
open boolean Controls visibility of the modal.
headerTitle string Sets the modal header title.

Outputs

Name Description
close Emitted when user closes the modal.

Custom Styling

You can customize the modal using CSS variables: 

rp-modal  { 
  --rp-modal-backdrop: rgba(0, 0, 0, 0.5);
  --rp-modal-width: 500px;
  --rp-modal-radius: 8px;
  --rp-modal-padding: 16px;
}

Example:

<rp-modal class="custom-modal">...</rp-modal>
